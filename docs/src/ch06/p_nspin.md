### Parameter: nspin

!!! warning

    Please DO NOT modify it.

**Definition**:

Number of spin projection.

**Type**

Integer

**Default value**

2

**Component**

Only for the **JASMINE** component.

**Behavior**

Determine the size of involved arrays.

**Comment**

See [nband](p_nband.md), [norbs](p_norbs.md), and [ncfgs](p_ncfgs.md) parameters.
